# MarkFlow

A beautiful, local-first Markdown editor with real-time preview and AI assistance.

Built by [Teda.dev](https://teda.dev), the AI app builder for everyday problems.

## Features

- **Live Preview**: See your changes instantly as you type.
- **Local AI**: Summarize and improve your writing using in-browser LLMs (WebLLM).
- **Privacy First**: No data is sent to any server. Everything runs locally.
- **Export**: Download your work as `.md` files.
- **Auto-save**: Your work is saved to local storage automatically.

## Tech Stack

- HTML5
- Tailwind CSS (CDN)
- jQuery
- Marked.js (Markdown Parsing)
- WebLLM (Client-side AI)

## Usage

1. Open `index.html` to see the landing page.
2. Click "Launch App" to enter the editor.
3. Type Markdown on the left, see the result on the right.
4. Enable AI features to get writing assistance (requires WebGPU compatible browser).
